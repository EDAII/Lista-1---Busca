### Instruções de uso:

 - No terminal, vá até o diretório que contém as pastas "inc" e "src"
 - Digite os seguintes comandos:

  ```bash
    $ make mkdir
    $ make
  ```

  - Para executar o código, digite:

  ```bash
    $ make run
  ```

  - Caso deseje limpar as pastas "bin" e "obj", digite:

  ```bash
    $ make clean
  ```
